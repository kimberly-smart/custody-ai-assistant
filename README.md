# CustodyAI Assistant

CustodyAI Assistant is a co-parenting application for divorced or separated parents. The goal is to help users review custody-related documents, draft calmer co-parenting messages, and keep organized parenting records such as health updates, medicine logs, exchanges, and communication notes.

## Live Demo

Frontend: https://custody-ai-assistant.vercel.app  
Backend API Docs: https://custody-ai-assistant-1.onrender.com/docs

## Project Status

This project is currently in early development and has a deployed demo version.

## Current Features

- Upload custody-related PDF documents
- Extract text from text-based and scanned PDF documents using OCR
- Ask questions based on uploaded documents
- Generate document-grounded answers with source references
- Summarize uploaded documents through natural language requests
- Rewrite emotional messages into calmer co-parenting communication

## Planned Features

- Maintain structured parenting, health, medicine, and exchange logs
- Generate organized records for personal documentation
- Add user authentication
- Add persistent cloud storage for uploaded documents
- Add chat history and follow-up refinement
- Add document selection for more precise Q&A

## Tech Stack

- Vite

- Tesseract OCR
- Docker
- Render
- Vercel
- SQLite for demo storage

## Important Note

This application is not intended to provide legal advice. It is designed to help users organize information, understand their uploaded documents in plain language, and communicate more clearly. Users should consult an attorney for legal decisions.
